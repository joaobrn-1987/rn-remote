🐧 Agente Linux

🐧 Ubuntu / Debian

Requer Ubuntu 20.04+ ou Debian 11+. Execute como root.

Instalação
curl -fsSL https://rnremote.joaoneto.tec.br/static/agent/install-ubuntu.sh -o /tmp/setup.sh && sudo bash /tmp/setup.sh
Remoção
curl -fsSL https://rnremote.joaoneto.tec.br/static/agent/uninstall-ubuntu.sh -o /tmp/remove.sh && sudo bash /tmp/remove.sh
↓ install-ubuntu.sh↓ uninstall-ubuntu.sh

🐧 CentOS Stream 9 / RHEL 9

Requer CentOS Stream 9 ou RHEL 9. Execute como root.

Instalação
curl -fsSL https://rnremote.joaoneto.tec.br/static/agent/install-centos.sh -o /tmp/setup.sh && sudo bash /tmp/setup.sh
Remoção
curl -fsSL https://rnremote.joaoneto.tec.br/static/agent/uninstall-centos.sh -o /tmp/remove.sh && sudo bash /tmp/remove.sh
↓ install-centos.sh↓ uninstall-centos.sh

🔥 Agente pfSense

Instalação

Requer pfSense 2.7+ (FreeBSD 14). Execute como root no pfSense.

Script de instalação
curl -fsSL https://rnremote.joaoneto.tec.br/static/agent/install-pfsense.sh | sh
Script de remoção
curl -fsSL https://rnremote.joaoneto.tec.br/static/agent/uninstall-pfsense.sh | sh
↓ agent-pfsense.py↓ install-pfsense.sh↓ uninstall-pfsense.sh

🪟 Agente Windows

Instalação

Requer Windows 10/11 com Python 3.8+. Execute o PowerShell como Administrador.

Instalação automática (PowerShell como Admin)
irm https://rnremote.joaoneto.tec.br/static/agent/install-windows.ps1 | iex
Remoção (PowerShell como Admin)
irm https://rnremote.joaoneto.tec.br/static/agent/uninstall-windows.ps1 | iex
